Installere ekstra funksjonalitet på Urørt: Greasemonkey utvidelse for Firefox

For å installere Greasemonkey følger du disse stegene:

  1. Gå til Gresemonkeys installasjonsside og følg resten av instruksene under. Siden vil mest sansynlig bli åpnet i en ny fane. For å komme tilbake til denne installasjonsinstruksen klikker du simpelthen på fanetittelen for denne siden.
  2. Klikk Add to Firefox for å starte utvidelseinstallasjonen.
  3. Etter 3 sekunder kan man klikke på Installer nå for å starte installasjonen.
  4. For å ta i bruk Greasemonkey må man starte Firefox på nytt. Klikk Start Firefox på nytt for å gjøre dette umiddelbart.
  5. Hvis du blir bedt om å bekrefte lukking av faner gjør du dette ved å klikke på Lukk faner
  6. Etter omstart av Firefox er du nå klar for å installere tillegg som gir deg ny funksjonalitet på Urørt.

Nå du er ferdig med å installere Greasemonkey er du klar for å installere tillegg som gir deg ny funksjonalitet på Urørt.
